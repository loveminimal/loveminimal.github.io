简介

![[assets/Pasted image 20250814141656.png]]

声笔飞系 是指飞单、飞码、飞延和飞讯。
单字基于飞单，飞码、飞延和飞讯只是词组的打法不同，但表着各自的词组体验倾向。
飞码打词方便，飞延、飞讯强调字词连打的韵律感。

飞系单字打法是：声母＋部首＋笔画＋笔画，
飞系不打部首便是「 声笔简码 」

飞系和简码互为一体，打字可以混用。

声笔简码

![[assets/Pasted image 20250814143804.png|300]]

使用笔画或满四码后，即可顶功，省略空格

单字：声母 ＋ 前三笔画。如：「 从 」cuou ，「 爱 」vuo
二字词：各字声母＋次字前两笔画。如：「 高兴 」gxoo
三字词：各字声母＋第三字首笔画。如：「 输入法 」srfo
多字词：前三字声母＋末字声母。如：「 床前明月光 」cqmg

声笔飞单

字持分成左右或上下，按先后顺序取部首。独体字取首笔画作为部首。

![[assets/41e9bddcfbbdc47cc004f12fb7c176df.png]]

上图中：

  • 黑色部首取声；
  • 蓝色部首为象形（形状接近/类似其字母的形）；
  • 紫色为括号内的声母，如括号内有两个字，则取深紫色字的声母；
  • 绿色部首需要单独记忆，如 V 中有很多是 ü 音的。

声笔飞单打法：声母＋部首＋部首以外的第一、二笔画，如：「码」msaa ； 「九」jpaa

笔画不够重复末笔，比如「九」字。

注意，如果单字为独体字，那么就取首笔作为部首，且编码用 hspdz 。

🪧我们来总结一下吧~

声笔飞单，声部一二。
部优左上，独体取首。
首取不同，h s pdz。
黑部为音，蓝部形近。
紫色为声，绿色强记。

声笔飞讯

打单字为主，只有三字词及以上才打词组 → 声笔飞讯

二字词：首字前两码＋次字首码＋次字前两笔画。如：「简码」jpmeu；「一直」yhzei
三字词：各字首码＋末字前两笔画。「计算机」jsjei
多字词：前三字首码＋末字首两笔画。「呼风唤雨」hfhei
成语：前三字首码＋末字大写首码。「呼风唤雨」

声笔飞码

打词组 → 声笔飞码

二字词：各取单字前两码。如：「不能」bhns；「赞颂」zbsy
三字词：前两字首码＋第三字前两码。如：「计算机」jsjm
多字词：前三字首码＋末字首码。如：「丰富多彩」ffdc

声笔飞延

打单为主，取词规则与飞码相同。

临时调用飞码时，只需末尾编码大写，或加 TAB 键自动转换。如：「丰富多彩」ffdC 或 ffdc+TAB

如何选择

这些声笔子方案，该怎么选呢？

只打单字 → 声笔飞单
打词组 → 声笔飞码
打单字为主，只有三字词及以上才打词组 → 声笔飞讯、声笔飞延（🔥）

打字日常聊天为主，不涉及专业领域术语，只想简易上手 → 声笔简码

声笔在标准打法之外，提供更多元的可选打法。比如缩减码、组合变换和增强模式。这是独立的，方便你更便捷打字，更是声笔对闲置键位的充分运用。当你愈加熟悉声笔，你会自然地想要去使用它们，提高你的效率。这让声笔能够更好地服务高阶熟练用户。

刚开始的时候，我们是不用关心这种进阶打法的~

魔改篇

我的声笔魔改 ➭ loveminimal/rime-sb ~

魔改了哪些内容呢？它：

基本功能跟随源仓库，增加额外定制功能，如下：
- 新增魔法声笔指法支持，许见 patch.yaml/algebra_magic
- 新增声笔英文方案 sben.schema.yaml
- 增加快符插件 quick_symbol.lua
- 自定义文件修改为 patch.txt
- 引入新的主题 style.yaml
- 添加 AHK 支持
- ……

魔法声笔指法是什么？它：「 ✓ 合并在全局 patch.yaml 中共享使用 」

爱护小手指，添加并击支持，如下：
- 并击组合直接输出大写字母「 空明码指法 」
- 小写字母提供并击组合：
  - we → q  sd → a  xc → z  io → p  
  - kl → ;  er → ！ ui → ？ cv → （  m, → ）
  - sf | jl → '
  - df | jk → 空格
  - fg | hj → 缩进
  - xv → ……  m. → ——

魔法声笔指法为什么而来呢？

- 为了减轻小手指的负担
- 为了方便键入大写字母（临时调用飞码）
- 同时兼容原声笔飞单的输入方式
- ……
