如何选择方案及配置

❶ 这么多的输入方案，都是什么？如何选择呢？

万象拼音分为两类：¹「 标准版 」、²「 辅助码版 」，有什么区别呢？

词典不同、部分预置功能不同。

万象  wàn xiàng 571                                                     → 标准版
万象  wàn;fp xiàng;wx  571                                              → 虎码辅助码
万象  wàn;av;ap;ag,ap;du;fp;dn;ap xiàng;du;dn;du,pd,ua;gh;wx;qj;en  571 → 多个辅助码

其中，标准版的词库就是基础，它和雾凇拼音、白霜拼音的词库格式是一样的，比后两者更强大之处在于万象的标准版词库是带声调的。

如果你是全拼用户，并且想保持既往的使用习惯，那么就选择标准版输入方案「 rime-wanxiang-base.zip 」。它默认是开启自动调频的（也就是多数用户理解的“智能造词”），会记下你的上屏词汇，以备后续输入。

如果你是 双拼用户，不了解、也不想了解“辅助码”是什么，也选择标准版输入方案「 rime-wanxiang-base.zip 」。

❷ 辅助码是什么呢？ 筛选候选词！如下，y 就是 辅 fǔ;yn 的辅助码（这里使用的是虎码首末辅助码）。

![[assets/Pasted image 20250529170207.png|230]] ![[assets/Pasted image 20250529173723.png|230]]

通过添加辅助码，实现了对候选词的进一步筛选，使得想要的词条排在靠前的位置，免去了翻页的烦恼。

辅助码好学吗？ 是的！万象拼音提供了 7 种类型的辅助码。如下：

--- 使用音托字根 ---
- 自然码辅助版本： rime-wanxiang-zrm-fuzhu.zip
- 墨奇辅助版本： rime-wanxiang-moqi-fuzhu.zip
- 小鹤辅助版本： rime-wanxiang-flypy-fuzhu.zip
- 简单鹤辅助版本： rime-wanxiang-jdh-fuzhu.zip
- 汉心辅助版本： rime-wanxiang-hanxin-fuzhu.zip

--- 利用形码字根 ---
- 五笔前二辅助版本： rime-wanxiang-wubi-fuzhu.zip
- 虎码首末辅助版本： rime-wanxiang-tiger-fuzhu.zip

如果你是五笔、虎码用户，直接使用后两个即可。

如果你是拼音用户，就选使用音托字根的辅助码版本，哪个都行。不知道怎么选？ 那就选 “自然码辅助版本” 。

自然码辅助码采用 “见字即根，部首优先” 的原则，比较符合用户直觉，是音托类字根拆分方案的代表。如 辅 fǔ;if ，把“辅”拆成了车i + 甫f ，再如 架 jià;mj,ml → 木m + 加j 或 木m + 力l 。你可以在另一篇文章 「 [[自然码辅助码入门教程]] 」 中了解更多，这里不再赘述。

问：“我是小鹤双拼用户，选择小鹤辅助码版本，可以吗？” 答：“当然可以！”

实际上随便哪个辅助码版本都是一样的，因为这些辅助码开发者在设计之初已经考虑了易用性。无论你选择哪一个，只需要稍微搜索一下相关的资料进行初步了解，然后在使用的过程中慢慢学习即可。

❸ 那个辅助码长长的是什么版本？

万象  wàn;av;ap;ag,ap;du;fp;dn;ap xiàng;du;dn;du,pd,ua;gh;wx;qj;en  571 → 多个辅助码

如果你是直接从仓库克隆的，其中的词典格式是这种“多个辅助码版本”。它唯一的“缺点”就是多个辅助码结合在一起，使得字典体积较大（当然我其实并为认为这是缺点）。

![[assets/Pasted image 20250627104311.png]]

它使得你可以在万象方案通过修改顶部配置来快速切换使用的辅助码类型，这在你前期想体验不同的辅助码类型是相当方便的。

❹ 小结一下

全拼、纯双拼       ➭ 标准版输入方案
                  rime-wanxiang-base.zip
双拼 + 单一辅助码  ➭ 任意辅助码版本（音托类推荐自然码辅助码版 ）
                  rime-wanxiang-zrm-fuzhu.zip

全拼、纯双拼                  
双拼 + 多辅助码    ➭ 仓库克隆（推荐新用户使用）同时包含标准版和辅助码版
                  git clone --depth=1 https://github.com/amzxyz/rime_wanxiang.git

安装

下载相应的配置包后，解压（zip）。

新用户可能会有 “配置放在哪儿？” 的疑惑，这里我们假设你已经安装了 Rime （如 Win 平台下的小狼毫）。

在 Windows 下，默认位置是在 C:\Users\你的用户名\AppData\Roaming\Rime ，你可以直接把解压后的文件夹重命名为 Rime 后，放在这个路径。

![[assets/Pasted image 20250627110920.png|150]] ![[assets/Pasted image 20250627111252.png|200]]

也可以通过 右键点击状态栏中的 “中”，再点击 程序文件夹 ，运行 WeaselSetup.exe 来指定配置路径。

![[assets/Pasted image 20250627111214.png|600]]

指定位置，选择路径，确定修改文件夹即可。

![[assets/Pasted image 20250627111944.png|325]]

如此，稍等片刻，你就可以使用万象拼音输入方案打字了。

安装后还有什么要做的吗？

可能。这里我们使用仓库克隆版本来说明（其他版本类似），它的默认配置是什么呢？

仓库克隆版同时包含标准版和辅助码版本，默认使用标准版，按下 Ctrl + ~ 如下：

![[assets/Pasted image 20250627112926.png|450]]

标准版（万象拼音）默认使用 全拼，同样的，你可以在 wanxiang.schema.yaml 中修改为想要使用的双拼方案，如“自然码”、“小鹤双拼” 等。

![[assets/Pasted image 20250627113352.png]]

辅助码版（万象拼音PRO）类似，比如这里我们修改为了 小鹤双拼 + 虎码首末辅助码 。

![[assets/Pasted image 20250627113732.png]]

修改后“重新部署”，即可。

3. 语法模型
✨适用类型： 所有版本皆可用
✨下载地址： wanxiang-lts-zh-hans.gram

语法模型是可选的，用来补充、修正词条候选，也是万象的较之于其它输入方案的一个特色。

方案中已包含应用设置，只需下载后，放在配置的根目录，重新部署，即可。
